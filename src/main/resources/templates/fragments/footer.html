About The Discussion Board

The Discussion Board is an online community where users can share blog posts and entertain discussions in the form of comments and replies and likes. This web application is proudly built on Java's Spring Framework. Some of the projects use from Spring Framework ecosystem of projects are Spring Boot, Spring MVC (Transactions, DAO, JDBC, ORM), Spring Security.

The project front-end is designed using Thymeleaf template engine, Bootstrap 4, and custom CSS, JavaScript and JQuery, Font Awesome, and Google Fonts. The backend business logic is built on core Java, Spring Inversion of Control (Dependency Injection). Data persistence is handled using Java Persistence API (JPA), Hibernate, and MySQL Database. Project dependencies are managed by Maven.
